# Supabase M-11D-6 辞退後の再申請復帰確認結果

作業日: 2026-06-01

## 1. 確認概要

M-11D-6として、ユーザー実ブラウザで本人申請辞退後の再申請復帰を確認した。

確認済み:

- 申請取り下げ後も、参加希望コメントは削除されず残る。
- 申請取り下げ後、申請中人数から除外される。
- 申請取り下げ後、mypageの参加申請中から対象セッションが消える。
- その後、参加希望コメントを投稿し直すと再申請扱いになる。
- `session_applications.status` は `canceled` から `pending` 相当に復帰する挙動として扱って問題ない。
- 公開版でも確認済み。

## 2. 再申請の扱い

本人申請辞退後の再申請は、専用UIや新規RPCを追加せず、既存の参加希望コメント投稿で行う。

これは既存方針の以下と整合する。

```text
コメント投稿 = 参加申請
```

`create_application_comment(target_session_id, comment_body)` は、本人の既存申請が `canceled` の場合に `pending` 相当へ戻すものとして扱う。

## 3. 申請人数カウント

再申請時にコメントは増えるが、申請人数はコメント件数ではなくユーザー単位で扱う。

そのため、同じユーザーが複数コメントしても、申請中人数は重複してカウントされない。

## 4. mypage表示

辞退後、mypageの参加申請中から対象セッションが消えることを確認済み。

再申請後は `pending` 相当へ復帰するため、既存のmypage取得条件では参加申請中の対象になる。今回の記録では、ユーザー確認済みの「辞退後に消えること」と「再申請が `pending` 相当に復帰すること」を正として扱う。

## 5. 公開版確認

公開版でも、辞退後にコメントが残り、人数カウントから外れ、再度コメント投稿することで再申請扱いに戻ることを確認済み。
