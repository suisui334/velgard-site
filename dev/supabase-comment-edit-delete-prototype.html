dev専用ページです。参加希望コメントの「編集」と「テスト用コメントの論理削除」だけを確認します。 本番ページには接続していません。

重要な注意

このページはプロトタイプDBのコメント本文と申請状態を変更します。本番の session-detail.html / calendar.html / 既存本番用 assets/js には接続していません。 close_session、Discord連携、通知、メール送信は扱いません。

入力するkeyはpublishable / anon keyのみです。service_role、secret key、DB password、Direct connection stringは絶対に使わないでください。 削除は論理削除ですが、この画面では元に戻しません。

確認手順

1. 接続するURLとkeyを手入力
2. ログインするテストユーザーで確認
3. セッションを選ぶ一覧から対象を選択
4. コメントを編集通常確認
5. テスト用だけ削除注意確認

通常確認
